# Travel Insurance - Static HTML Version

This is a pure HTML/CSS/JS implementation of the Travel Insurance application. logic.

## Project Structure
- `index.html`: Home page
- `quote.html`: Quote results page
- `payment.html`: Payment page
- `assets/`: Contains `css`, `js`, and `images`

## Running Locally
You can use any static file server.

### Using Node.js (npx)
```bash
npm start
# OR
npx http-server .
```

### Using Python
```bash
python -m http.server
```

## Deployment
This project is ready for Vercel, Netlify, or GitHub Pages.
- **Vercel**: Simply import this repository. It will automatically detect the static files.
